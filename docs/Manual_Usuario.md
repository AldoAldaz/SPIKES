# Manual de Usuario - SPIKES!

Bienvenido a **SPIKES!**, un desafío de precisión y reflejos.

## Objetivo del Juego

Pilota tu nave a través de un túnel infinito lleno de obstáculos mortales.
Tu objetivo es llegar lo más lejos posible y acumular la mayor puntuación.

## Sistema de Puntuación

El juego cuenta con un sistema de riesgo-recompensa:

- **Huecos Grandes (Verdes):** Otorgan 1 punto. Son seguros.
- **Huecos Medianos (Amarillos):** Otorgan 2 puntos. Requieren cuidado.
- **Huecos Pequeños (Rojos):** Otorgan 3 puntos. ¡Solo para expertos!

## Controles

- **JUGAR:** Haz clic en el botón "Jugar" del menú principal.
- **VOLAR:** Mantén presionado **Clic Izquierdo** o la **Barra Espaciadora** para elevarte. Suelta para descender por gravedad.
- **PAUSA/SALIR:** Presiona la tecla **ESC** en cualquier momento para cerrar el juego.
